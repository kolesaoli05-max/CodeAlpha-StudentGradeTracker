# Student Grade Tracker (Java)

## Why I Built This

I wanted to practice core Java concepts like OOP and data handling by building a simple but structured system to manage student records.

## What Makes This Different

* Roll numbers are enforced as **unique identifiers**
* Data is protected using **encapsulation** (no direct access to fields)
* Updates are controlled through specific methods instead of direct modification
* Input validation is handled without overcomplicating the logic

## Features

* Add a new student
* Delete a student
* Update name or grade
* Search student by roll number
* Display all records
* Show statistics (average, highest, lowest)

## Tech Used

* Java
* ArrayList
* Object-Oriented Programming

## How to Run

javac StudentGradeTracker.java

java StudentGradeTracker

## What I Learned

* How to design classes using encapsulation
* How to manage data using ArrayList
* How to structure a menu-driven program
* Importance of validation and data integrity
